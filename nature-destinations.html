Where to go — 15 nature destinations within reach

This page is a menu, not an itinerary. Five of these are already locked into the schedule on the Itinerary page (marked with a gold ✓ Locked badge). The other ten are alternates — swap any in, swap any out. Filtered to walks + easy hikes only. Sunset rating is graded 1-3 — three suns means the marquee reason to drive there.

Legend

  • 🚶 walk — flat, <30 min, no elevation. 🥾 easy hike — mild grade, <1.5h, no scrambling. Anything harder is cut.

How to use this menu

  • Default plan: stick with the 5 Locked spots in the Itinerary. They're the strongest hits of each region.
  • Swap an alternate in: any non-locked card in the same region pairs naturally with the existing day. Example: swap Hintersee (Ramsau) into the Königssee day instead of the Obersee leg if the boat round-trip feels too long.
  • Add a "drive day": Krimml Waterfalls or Grossglockner are full-day commitments. Each replaces a slower lake-day at the apartment if you want one big alpine drive.
  • Slovenia call: Lake Bled + Vintgar = an overnight in a different country. Currently OUT. If yes, we re-plan the back half — talk first.
  • Leave a note on the floating 💬 button if you want any of these moved into the locked plan.
