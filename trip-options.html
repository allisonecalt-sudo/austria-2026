Trip shape — how we got to 4 bases, no summit overnight

The trip shape went through 4 versions. Each one is below with the structure we shipped, why it changed, and a link to the decisions-log page that captured the call. Current shape = Salzburg → Zell am See → Gosau → Airport. No summit sleep.

This page used to be a 3-shape picker (Apt-Jezero vs Splurge vs Quiet-Gems, all with a locked Schafbergspitze summit overnight). After Avital's Mon May 18 counter-proposal + the Lodge am Krippenstein availability collapse, the trip restructured. The picker retired; this page is now the honest narrative of how the shape evolved. Pullable archives (per the rule): every version below links to the page that holds its full detail.

How to read this page

  • Top to bottom = chronological. v1 is the first shape we built (May 16-17); v4 is the live shape (May 19).
  • Each beat links to its archive page. The deep call notes, prices, alternate-property research, and Playwright availability checks all live on the linked page so this page can stay a narrative.
  • "What changed" is honest. Reviews, availability, Avital's voice notes — what actually moved the shape each time. No spin.
  • Current shape detail lives on bases.html. The 7-day-by-day is on itinerary.html. Lodging hub is stay.html.

Sources for the timeline: Booking.com live listings, Playwright availability re-checks 2026-05-17 + 2026-05-18, Avital's WhatsApp voice notes (Sun May 17 23:25 + 23:26), and her 4 austria-notes app comments (Sun May 17 evening). Full event log in second-brain/archive/chronicle/2026-05-18-trip-site-overnight-arc.md.

What's NOT here: flights, car rental, food, gas, attraction tickets, incidentals. See Costs page for the full accounting.
