The geography

4 bases, 3 moves. Salzburg (Fri-Sun) → Obertraun mountain anchor (Sun-Wed, the base for the lake-cluster day trips) → Schafbergspitze 1,783m summit (Wed-Thu, the sleep-above-13-lakes night) → Salzburg-airport (Thu-Fri). Most of the day-trip targets are reachable in under 1h15m from the Obertraun anchor.

If you go for these places, sleep here

The lodging map matched to the geography. The v1 plan groups two clusters into two stays — Salzburg for the Berchtesgaden + Hohe Tauern reach, Obertraun for the Salzkammergut anchor. Each cluster below shows the curated picks already on the Stay page.

Cluster 1 · Salzkammergut

Lakes + Dachstein + cog rail

Hallstatt, Krippenstein, Gosausee, Schafbergspitze, Wolfgangsee, Attersee — six of the seven Salzkammergut spots are inside 65 min of Obertraun.

Sleep at:

Obertraun / Hallstatt-area apartments — 3 nights (Sun-Wed), the deep mountain anchor. Wed-night base shifts to Berghotel Schafbergspitze 1,783m via the Schafbergbahn cog from St. Wolfgang — the locked unique-experience summit overnight.
Pick: Haus Edelweiss · €142/night.
Backup: Ferienhof Osl (working farmhouse) · €136/night.
Bad Goisern option: Austrian Apartments · €136/night.
See all 6 options →

Cluster 2 · Berchtesgaden

Königssee + Bavarian Alps day trips

Königssee, Hintersee, Almbachklamm — all under 35-50 min from Salzburg. From the Obertraun anchor, Königssee is still doable as a single big day (1h15m each way for the Tuesday peak moment).

Sleep at:

Salzburg city / Linzergasse for Shabbat (Fri-Sun) — the Berchtesgaden day-trip distances also work from Salzburg if a fork ever wanted to extend a Salzburg leg.
Pick: master Linzergasse · €128/night · 5-min walk to Chabad.
See all 6 options →

Cluster 3 · Hohe Tauern / Pongau

Werfen, ice cave, gorges, alpine road

Eisriesenwelt, Liechtensteinklamm, Krimml, Grossglockner — all 30-90 min from Salzburg. Werfen is locked into the Thursday transit day from Hallstatt back toward the airport (1h15m from Obertraun via Werfen).

Sleep at:

Salzburg base works for any of these as half-day swap-ins. No separate cluster lodging needed — the v1 plan threads Werfen on the Thursday transit. Liechtensteinklamm + Krimml + Grossglockner are full-day "drive day" swaps that work from either base. See Salzburg options →
