The four bases — pick the mountain anchor

The trip has 4 bases: Salzburg → Mountain anchor → Schafbergspitze summit → Airport. Three of those four are locked. The one decision left is which mountain anchor for the Sun-Wed midweek. 3 fully-built alternates below, each with drive matrix to all 21 nature spots, lodging picks, daily flow + a "why we'd lean here" per config. Avital's pick: Obertraun (closest to the lake, shortest drives, kitchen-y vibe). Berchtesgaden + St. Wolfgang are kept here as honest swap-ins.

The 4-base shape: Salzburg Shabbat anchor (Fri-Sun, 2 nights) · Mountain anchor (Sun-Wed, 3 nights — Obertraun by default, alternates below) · Berghotel Schafbergspitze 1,783m summit overnight (Wed-Thu, 1 night) · airport-side last night (Thu-Fri, 1 night). The only open choice is the 3-night mountain anchor (Sun-Wed) that shapes the midweek. Below are 3 fully-developed mountain-anchor alternates — each with its drive matrix to all 21 nature destinations, lodging picks, daily flow, pros + cons, and cost delta. Salzburg, Schafbergspitze, and the airport night are the same across all 3 alternates (not optional).

🏔 Wed Jul 29 night across all 3 configs: Berghotel Schafbergspitze at 1,783m, accessed via Schafbergbahn cog railway from St. Wolfgang am Wolfgangsee. Last cog UP ~17:00, sunset on the summit, sleep above 13 lakes, first cog DOWN ~09:00 Thursday. BOOK direct ~2-3 weeks ahead — small hotel (~34 ppl capacity). 📞 +43 6138 35 42 · 🌐 schafberg.net

How to use this page

  • Read the 3 pitches first — one paragraph each, with a "why we'd lean here" line per config. Then pick the 2 you want to compare in detail.
  • Drive matrix tells the geography truth: green = at your door (≤10 min), yellow = easy (≤30), orange = day trip (≤60), red = long day (60+). Avital's ceiling is "long" so red rows are honest skip-flags, not challenges.
  • Lodging picks per config are real Booking.com listings with vibe tags + bedrooms/beds + laundry status. 2-BR / 2-sleeping-area picks sorted to top for the main 3-night mountain anchor (Sun-Wed).
  • No "correct" answer. The three are honest trade-offs: closest-to-lake (Obertraun) / closest-to-Königssee (Berchtesgaden) / closest-to-Schafberg (St. Wolfgang). We lean Obertraun because Königssee is a one-day trip either way and Schafberg is built-in via the cog night — but if Avital wants to wake up looking at the Watzmann, Berchtesgaden is right.
  • Drop a note on the 💬 floating button if you want to lock a config or have a question.
